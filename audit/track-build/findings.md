## B.5 — Tooling Gaps

- **Tailwind version mismatch.** `package.json:31` says `^3.4.19`. README claims v4. Components use `bg-emerald-500/20`, `text-${color}-400` (dynamic class generation) — **these break in Tailwind 4's stricter JIT** but work in 3.4.
- **`@supabase/supabase-js` in deps but unused.** Both apps list it; neither imports it. Dead dep.
- **No `.npmrc` or `engines` field.** Node version is unenforced.
- **No lockfile?** `package-lock.json` is generated by `npm install` (after the install ran). Not committed in original clone.
- **Recharts deprecated.** `npm warn deprecated recharts@2.15.4` — should bump to v3 (breaking changes).
- **`/admin-portal/--store-dir` symlink in repo root.** Looks like an accidentally-committed symlink. Not referenced in build.
